# 🛒 Supermarket Retail Strategy and Analysis Dashboard

An interactive and data-driven Power BI dashboard designed to analyze supermarket retail performance across sales, customers, stores, and brands.  
This project transforms raw retail transaction data into actionable business insights using powerful visualizations, KPIs, and dynamic filtering.

---

# 📌 Project Objective

The objective of this project is to:
- Analyze retail sales performance
- Track customer purchasing behavior
- Identify top-performing brands and stores
- Monitor sales trends over time
- Provide business insights for better decision-making

This dashboard helps stakeholders quickly understand key retail metrics and performance indicators.

# 📂 Dataset Information

The dataset contains supermarket retail transaction data including:
- Sales Amount
- Product Categories
- Brands
- Customer Types
- Store Numbers
- Quantity Sold
- Transaction Metrics
- Sales Dates

Dataset File:
- `retail_dataset_25k.xlsx`

# 📊 Dashboard Overview

The report contains **3 interactive dashboard pages**:

---

# 🏠 1. Retail Analysis Home

This page provides an overall business performance summary.

### Key Features
- Total Sales KPI
- Total Customers KPI
- Product Quantity Sold
- Transaction per Customer
- Average Price per Unit
- Average Sales per Customer

### Visual Analysis
- Monthly Sales Trend
- Sales by Brand
- Product Category Performance
- Customer Type Distribution

### Business Insights
- Chips category generated the highest revenue
- Premium customers contributed strong sales
- Sales remained relatively stable throughout the year

---

# 🏬 2. Store Analysis

This dashboard focuses on store-level performance analysis.

### Key Features
- Store-wise Total Sales
- Product Quantity by Store
- Top 5 Performing Stores
- Bottom 5 Performing Stores
- Transaction Metrics by Store

### Business Insights
- Certain stores consistently outperformed others
- Some stores showed lower transaction efficiency
- Product sales varied significantly between stores

### Analytical Benefits
- Helps identify high-performing locations
- Supports operational improvement decisions
- Enables store comparison analysis

---

# 🏷 3. Brand Analysis

This page analyzes brand performance and product contribution.

### Key Features
- Brand-wise Sales Analysis
- Product Quantity Sold by Brand
- Top 5 Brands
- Bottom 5 Brands
- Average Product Pricing

### Business Insights
- Kettle and Pringles were among the top-performing brands
- Lower-performing brands contributed less to total revenue
- Brand pricing impacted sales volume

### Analytical Benefits
- Helps identify profitable brands
- Supports inventory and marketing decisions
- Tracks brand contribution to overall business growth

---

# 🎯 Key KPIs Used

| KPI | Description |
